🚀 Blitztalk FE Chat

A modern real-time chat frontend built for seamless communication, speed, and clean UI — designed to integrate with a Socket.IO-powered backend.

✨ Features
⚡ Real-time messaging (Socket.IO ready)
💬 Clean and responsive chat UI
👤 User-friendly message flow
🔄 Live updates without refresh
📱 Works across devices (responsive design)
🔌 Easy backend integration
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Realtime Engine: Socket.IO (client)
Version Control: Git + GitHub
📂 Project Structure
Blitztalk-FE-Chat/
│── index.html        # Main UI
│── style.css         # Styling
│── script.js         # Chat logic
│── assets/           # Images / icons (if any)
⚙️ Setup & Installation
Clone the repository
git clone https://github.com/amanraula/Blitztalk-FE-Chat.git
Open project folder
cd Blitztalk-FE-Chat
Run locally
Just open index.html in your browser
🔌 Backend Connection

Make sure your backend server (Socket.IO) is running.

Update this line in your JS file if needed:

const socket = io("http://localhost:3000");
🧠 How It Works
User enters message
Message sent via Socket.IO
Server broadcasts to all users
UI updates instantly
